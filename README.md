# Posts and comments app
This project is little clone of reddit post and comments functionality.

## Commits convention

Commit types

📃: Documentation
🐛: Bug fix
🎨: Style
🔁: Refactor
🔧: Config

Commit scopes

🐋: Docker
🔌: Backend
🖥️: Frontend
🗄️: Database
🧰: Admin

Commit structure

```
<type>(<scope>): <subject>
```

Example commit

```
📃: Add commit convention to README.md
🐛(🔌): Fix bug in post creation
```
